# HashCare

HashCare Рђћ рЃћрЃА рЃљрЃарЃўрЃА рЃўрЃюрЃЮрЃЋрЃљрЃфрЃўрЃБрЃарЃў рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃљрЃдрЃарЃўрЃфрЃ«рЃЋрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃЎрЃарЃўрЃърЃбрЃЮрЃЋрЃљрЃџрЃБрЃбрЃўрЃА рЃЏрЃљрЃўрЃюрЃўрЃюрЃњрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃЌрЃў рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃА рЃърЃџрЃљрЃњрЃўрЃюрЃўрЃА рЃАрЃљрЃ«рЃўрЃЌ.

## ­Ъџђ HashCare-рЃўрЃА рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃћрЃЉрЃў

Рђб **­ЪЊі рЃЏрЃљрЃўрЃюрЃўрЃюрЃњрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃўрЃА рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃў:**

рЃЋрЃЮрЃарЃЦрЃћрЃарЃћрЃЉрЃўрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃА, рЃърЃБрЃџрЃўрЃА рЃ░рЃћрЃерЃарЃћрЃўрЃбрЃўрЃАрЃљ рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЏрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃЌрЃЋрЃљрЃџрЃДрЃБрЃарЃўрЃА рЃЊрЃћрЃЋрЃюрЃћрЃЉрЃљ.

Рђб **­ЪЊІ рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃљрЃдрЃарЃўрЃфрЃ«рЃЋрЃљ:**&#x20;

рЃЏрЃљрЃўрЃюрЃўрЃюрЃњрЃерЃў рЃЕрЃљрЃарЃЌрЃБрЃџрЃў рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ рЃЊрЃљ рЃљрЃюрЃљрЃџрЃўрЃќрЃў.&#x20;

Рђб **­Ъћї рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃўрЃА рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃў рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ:**&#x20;

рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃЏрЃўрЃћрЃарЃЌрЃћрЃЉрЃљ рЃЊрЃљ рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃА рЃЏрЃЮрЃарЃњрЃћрЃЉрЃљ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃљрЃЊ.&#x20;

Рђб **РЈ▒№ИЈ рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў (RT):**

рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃАрЃљ рЃЊрЃљ рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ рЃљрЃЦрЃбрЃБрЃљрЃџрЃБрЃарЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў.&#x20;

Рђб **­ЪЊѕ рЃљрЃюрЃљрЃџрЃўрЃбрЃўрЃЎрЃљ рЃЊрЃљ рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃў:**&#x20;

рЃЕрЃљрЃерЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃљрЃюрЃљрЃџрЃўрЃбрЃўрЃЎрЃБрЃарЃў рЃ«рЃћрЃџрЃАрЃљрЃгрЃДрЃЮрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃЮрЃЉрЃўрЃА рЃњрЃљрЃАрЃљрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃћрЃЉрЃџрЃљрЃЊ.

## РЮЊрЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА HashCare?

рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃў рЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃЊрЃћрЃЉрЃљ рЃљрЃњрЃћрЃюрЃбрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃљрЃЏрЃЮрЃфрЃљрЃюрЃћрЃЉрЃА:&#x20;

Рђб рЃЏрЃўрЃћрЃарЃЌрЃћрЃЉрЃБрЃџ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃћрЃЉрЃќрЃћ рЃЏрЃћрЃбрЃарЃўрЃЎрЃћрЃЉрЃўрЃА рЃерЃћрЃњрЃарЃЮрЃЋрЃћрЃЉрЃљ. \
Рђб рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃфрЃћрЃЏрЃљ рЃфрЃћрЃюрЃбрЃарЃљрЃџрЃўрЃќрЃћрЃЉрЃБрЃџ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃерЃў рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА.&#x20;

HashCare рЃњрЃЌрЃљрЃЋрЃљрЃќрЃЮрЃЉрЃЌ:&#x20;

Рђб ­ЪјЏ№ИЈ **рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃћрЃЉрЃљрЃЊрЃў рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў** Рђћ рЃЏрЃЮрЃўрЃарЃњрЃћрЃЌ рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃљрЃюрЃљрЃџрЃўрЃбрЃўрЃЎрЃБрЃарЃў рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃљрЃЊ.\
Рђб ­Ъџе **рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЋрЃџрЃћрЃюрЃљ** Рђћ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃарЃЮрЃБрЃџрЃў рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃфрЃўрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃарЃћрЃљрЃњрЃўрЃарЃћрЃЉрЃљ.&#x20;

HashCare Рђћ рЃћрЃА рЃљрЃарЃўрЃА рЃБрЃюрЃўрЃЋрЃћрЃарЃАрЃљрЃџрЃБрЃарЃў рЃ«рЃћрЃџрЃАрЃљрЃгрЃДрЃЮ рЃЏрЃљрЃўрЃюрЃўрЃюрЃњрЃўрЃА рЃЏрЃЌрЃћрЃџрЃў рЃърЃарЃЮрЃфрЃћрЃАрЃўрЃА рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃљрЃдрЃарЃўрЃфрЃ«рЃЋрЃљрЃАрЃљ рЃЊрЃљ рЃљрЃюрЃљрЃџрЃўрЃќрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃЏрЃўрЃАрЃў рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ рЃЊрЃљ рЃдрЃарЃЏрЃљ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃА рЃ«рЃЊрЃўрЃА рЃДрЃЋрЃћрЃџрЃљ рЃЊрЃЮрЃюрЃўрЃА рЃЏрЃљрЃўрЃюрЃћрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.
