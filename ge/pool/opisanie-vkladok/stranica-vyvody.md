# ­ЪњИ рЃњрЃЋрЃћрЃарЃЊрЃў РђюрЃњрЃљрЃЏрЃЮрЃбрЃљрЃюрЃљРђЮ

рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ РђюрЃњрЃљрЃЏрЃЮрЃбрЃљрЃюрЃљРђЮ рЃюрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃўрЃљ рЃДрЃЋрЃћрЃџрЃљ рЃбрЃарЃљрЃюрЃќрЃљрЃЦрЃфрЃўрЃўрЃА рЃўрЃАрЃбрЃЮрЃарЃўрЃљ, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃЌрЃљрЃюрЃ«рЃўрЃА рЃњрЃљрЃЏрЃЮрЃбрЃљрЃюрЃљрЃАрЃЌрЃљрЃю.

Рђб рЃЏрЃўрЃюрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃЌрЃљрЃюрЃ«рЃљ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃбрЃљрЃюрЃљрЃЊ: рЃбрЃарЃљрЃюрЃќрЃљрЃЦрЃфрЃўрЃћрЃЉрЃў рЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃЊрЃћрЃЉрЃљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ, рЃарЃЮрЃњрЃЮрЃарЃф рЃЎрЃў рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЉрЃљрЃџрЃљрЃюрЃАрЃў рЃЏрЃўрЃљрЃдрЃгрЃћрЃЋрЃА 0.001 BTC-рЃА.

## ­ЪЊІ рЃфрЃ«рЃарЃўрЃџрЃўрЃА рЃерЃћрЃЏрЃљрЃЊрЃњрЃћрЃюрЃџрЃЮрЃЉрЃљ РђюрЃњрЃљрЃЏрЃЮрЃбрЃљрЃюрЃљРђЮ

\
рЃфрЃ«рЃарЃўрЃџрЃў рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃАрЃЋрЃћрЃбрЃћрЃЉрЃА:

Рђб ­ЪЊЁ рЃЌрЃљрЃарЃўрЃдрЃў рЃЊрЃљ рЃЊрЃарЃЮ Рђћ рЃЌрЃљрЃюрЃ«рЃўрЃА рЃњрЃљрЃЏрЃЮрЃбрЃљрЃюрЃўрЃА рЃЏрЃЮрЃЏрЃћрЃюрЃбрЃў.

Рђб ­Ъњ░ рЃЌрЃљрЃюрЃ«рЃљ Рђћ BTC-рЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЊрЃљрЃарЃўрЃфрЃ«рЃБрЃџрЃўрЃљ рЃбрЃарЃљрЃюрЃќрЃљрЃЦрЃфрЃўрЃўрЃА рЃцрЃљрЃарЃњрЃџрЃћрЃЉрЃерЃў.

Рђб ­ЪЈд рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў Рђћ рЃАрЃљрЃцрЃБрЃџрЃўрЃА рЃюрЃЮрЃЏрЃћрЃарЃў, рЃарЃЮрЃЏрЃћрЃџрЃќрЃћрЃф рЃњрЃљрЃЊрЃљрЃўрЃњрЃќрЃљрЃЋрЃюрЃљ рЃЌрЃљрЃюрЃ«рЃљ.

Рђб ­ЪћЌ рЃбрЃарЃљрЃюрЃќрЃљрЃЦрЃфрЃўрЃўрЃА рЃюрЃЮрЃЏрЃћрЃарЃў Рђћ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃбрЃЮрЃарЃў рЃЉрЃџрЃЮрЃЎрЃЕрЃћрЃўрЃюрЃерЃў.
